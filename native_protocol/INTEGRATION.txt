# ── Save file format (for reference / hand-testing) ────────────────────────
# Create a file like /ext/subghz/my_gate.sub with this content:
#
#   Filetype: Flipper SubGhz Key File
#   Version: 1
#   Frequency: 318000000
#   Preset: FuriHalSubGhzPresetOok650Async
#   Protocol: Allstar Firefly
#   Key: +--000++-
